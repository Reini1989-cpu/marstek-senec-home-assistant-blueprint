# Sicherheit

Bitte veröffentliche in Issues, Pull Requests oder Beispieldateien keine:

- Home-Assistant-Tokens oder Kennwörter
- Inhalte aus `secrets.yaml`
- interne IP-Adressen, Hostnamen oder Zugangsdaten
- vollständigen Konfigurations- oder Backup-Archive

Wurde ein Geheimnis versehentlich veröffentlicht, sollte es sofort widerrufen beziehungsweise geändert werden, bevor die Repository-Historie bereinigt wird.
